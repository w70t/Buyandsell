# سجل المهام — Souqna TASKS

حالة البناء الحالية مقابل المتطلبات، وما يليها. (خارطة الميزات المستقبلية التفصيلية
بمعايير القبول في `ROADMAP.md`.)

## ✅ منجز ومُتحقق منه (pytest + متصفح)

### الأساس
- [x] Backend FastAPI (async) + PostgreSQL 16 + مهاجرات Alembic تلقائية عند الإقلاع
- [x] Docker Compose واحد للراسبيري وللـ VPS (arm64/amd64) + nginx وكيلاً عكسياً
- [x] كل الإعدادات عبر `.env` (12-factor) — لا قيم مثبّتة في الكود
- [x] تخزين صور قابل للتبديل: قرص محلي ⇠⇢ MinIO/S3 بمتغيّر واحد
- [x] Redis اختياري لحدود الطلبات عند التوسّع (`--profile redis`)

### الميزات
- [x] حسابات: تسجيل/دخول برقم عراقي، JWT مع تدوير، جلسة دائمة
- [x] إعلانات: نشر/تعديل/حذف/حالات (نشط/مباع/مخفي)، صور متعددة معاد ترميزها، عدّاد مشاهدات
- [x] بحث وفلاتر: نص + قسم + محافظة + مدى سعر + فرز (أحدث/أرخص/أغلى) + ترقيم
- [x] مفضلة مع إشعار للمالك
- [x] محادثات بائع/مشترٍ مع علامات قراءة وعدّادات غير مقروء
- [x] إشعارات داخل المنصّة (رسالة/مفضلة/إشراف) + عدّاد في الواجهة
- [x] إشعارات تيليغرام للمشرف (مستخدم جديد/إعلان جديد/بلاغ جديد) — اختيارية
- [x] بلاغات وإشراف: أسباب مصنّفة، منع التكرار، **إخفاء تلقائي بعد N بلاغات**، طابور معالجة
- [x] سجل تدقيق لكل إجراء إداري (بما فيه إجراءات النظام)
- [x] الموقع العام SSR بالعربية (RTL + خط Cairo مضمّن) — صديق لمحركات البحث
- [x] لوحة إدارة ويب: إحصاءات، مستخدمون (حظر/أدوار)، إعلانات، بلاغات، أقسام، تدقيق
- [x] أمان: Argon2id، rate limiting، CSRF للويب، رفع آمن، أدوار، مستخدم غير جذري
- [x] نسخ احتياطي (pg_dump + الصور) مع سياسة احتفاظ
- [x] اختبارات: 34 pytest + 16 تدفق متصفح Playwright — **الكل أخضر**
- [x] التوثيق العشري الكامل في `docs/`

## 🔜 التالي المقترح (بالأولوية)
1. **F1 إشعارات Push (FCM)** لتطبيق Flutter — أكبر رافعة احتفاظ (البنية الحالية للإشعارات جاهزة كمصدر).
2. **F2 توثيق OTP بالهاتف** — يقطع الحسابات الوهمية (يتطلب مزوّد SMS/WhatsApp).
3. ربط تطبيق Flutter بالمسارات الجديدة (إشعارات/بلاغات) — الـ API جاهز.
4. **F5 بحث عربي حقيقي** (PostgreSQL FTS + pg_trgm + تطبيع الهمزات).
5. **F6 خصائص ديناميكية لكل قسم** (سيارات: موديل/سنة/ممشى…).
6. محادثة لحظية WebSocket بدل الـ polling (F10).
7. أحجام صور متعددة WebP + BlurHash (F11).
8. تقييمات البائعين (F3) والإعلانات المميزة مع دفع محلي (F9).

- لا تُعتبر الميزة منجزة إلا بعد اختبار متصفح أخضر على المكدّس الكامل.
